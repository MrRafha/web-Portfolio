# Imagens dos Projetos

Coloque as imagens de cada projeto em suas respectivas pastas.

## Estrutura

```
projects/
├── ultimate-tracker/
│   ├── 01.png  (Visão geral)
│   ├── 02.png  (Fluxo de comunidade)
│   └── 03.png  (Escalabilidade)
├── ifala/
│   ├── 01.png  (Interface principal)
│   ├── 02.png  (Componente colaborativo)
│   └── 03.png  (Experiência final)
├── hello-kitty/
│   ├── 01.png  (Tela inicial)
│   ├── 02.png  (Fluxo de lembrete)
│   └── 03.png  (Componentes visuais)
└── bots/
    ├── 01.png  (Painel de comandos)
    ├── 02.png  (Integrações)
    └── 03.png  (Monitoramento)
```

## Instruções

1. **Renomear arquivos**: Use o padrão `01.png`, `02.png`, `03.png` para cada projeto
2. **Formato**: PNG é recomendado, mas também funciona com JPEG e WebP
3. **Tamanho**: Recomendado mínimo de 800x400px para boa qualidade
4. **Caminho automático**: As imagens serão carregadas automaticamente no componente

As imagens aparecerão na galeria esquerda de cada card do projeto no carrossel.
